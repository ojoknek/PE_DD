# DD結果

## 概要
デューデリジェンス（DD）の結果を集約管理するディレクトリです。すべてのDD結果はこのディレクトリ配下に統一して保存されます。

## ディレクトリ構成
- `nn_dd/`: NN DD結果（Non Name Sheetを基にした評価）
- `im_dd/`: IM DD結果（Information Memorandumを基にした評価）
- `additional_dd/`: 追加DD結果（人間による議論・調査）

## 設計思想
- **集約管理**: すべてのDD結果を一箇所に集約することで、結果の把握と管理を容易にする
- **明確な分離**: 入力資料（`nn/`, `im/`）とDD結果（`dd_results/`）を明確に分離
- **段階的な評価**: NN DD → IM DD → 追加DD という段階的な評価プロセスを反映

## 使用方法
各DDワークフロー実行後、対応するディレクトリに結果が自動的に保存されます：
- NN DDワークフロー実行後 → `nn_dd/` に保存
- IM DDワークフロー実行後 → `im_dd/` に保存
- 追加DDワークフロー実行後 → `additional_dd/` に保存

## 各ディレクトリの詳細
各ディレクトリの詳細は、それぞれのREADMEを参照してください。
