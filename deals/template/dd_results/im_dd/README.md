# IM DD結果

## 概要
Information Memorandum（IM）を基にしたIM DD評価の結果を管理します。

## ファイル構成
- `report.md`: IM DD結果レポート（人間が読みやすい形式）
- `data.json`: 構造化データ（機械処理用）

## 評価内容
- 定量情報と定性情報を統合した評価
- 事業性評価、財務評価、経営陣評価、リスク評価、投資条件評価の各論点の評価
- DCF分析、バリュエーション分析、リスク分析の実施
- 総合スコアと投資推奨度の算出

## 使用方法
1. IM資料を `../im/materials/` に配置
2. IM DDワークフローを実行（`agents/workflows/im_dd_workflow.md`を参照）
3. 結果がこのディレクトリに自動的に保存されます

## 関連ディレクトリ
- 入力資料: `../im/`
- 評価ロジック: `dd_logic/im_dd/`
