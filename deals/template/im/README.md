# Information Memorandum（IM）関連資料

## 概要
Information Memorandum（IM）を基にしたIM DDの入力資料を管理します。

## ディレクトリ構成
- `materials/`: 元資料（PDFファイル）
- `processed/`: 処理済みデータ（構造化されたデータ）

## 使用方法
1. IM資料を `materials/` に配置
2. IM DDワークフローを実行（`agents/workflows/im_dd_workflow.md`を参照）
3. 処理済みデータが `processed/` に保存されます
4. **DD結果は `../dd_results/im_dd/` に保存されます**

## 関連ディレクトリ
- DD結果: `../dd_results/im_dd/`
