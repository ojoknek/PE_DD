# IM DD結果

## 概要
Information Memorandum（IM）を基にしたIM DD評価の結果を管理するディレクトリです。IM DDワークフロー実行時に自動生成され、このディレクトリに保存されます。

## ファイル構成
- `report.md`: AIによるIM DD結果レポート（人間が読みやすい形式）
  - 評価結果のサマリー
  - 各論点の評価結果
  - 分析結果（DCF分析、バリュエーション分析、リスク分析）
  - 総合スコアと投資推奨度

## 評価内容
IM DDでは以下の評価が実施されます：
- **定量情報と定性情報を統合した評価**
- **各論点の評価**:
  - 事業性評価
  - 財務評価
  - 経営陣評価
  - リスク評価
  - 投資条件評価
- **分析の実施**:
  - DCF分析
  - バリュエーション分析
  - リスク分析
- **スコア算出**:
  - 総合スコア
  - 投資推奨度の算出

## 使用方法

### 1. 入力資料の配置
IM資料（PDF）を `../../im/materials/` に配置します。

### 2. ワークフローの実行
IM DDワークフローを実行します。
- 詳細な手順: `agents/workflows/im_dd_workflow.md` を参照
- AI/LLMを使用したプロンプトベースで実装
- 計算ロジックは数式仕様書（`dd_logic/im_dd/calculations/`）を参照

### 3. 結果の確認
結果がこのディレクトリに自動的に生成されます：
- `report.md`: 評価結果レポートを確認

## 評価論点
詳細な評価論点は以下を参照してください：
- `dd_logic/im_dd/evaluation_points/evaluation_points.md`: IM DD評価論点
