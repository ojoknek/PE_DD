# VDR（Virtual Data Room）関連資料

## 概要
VDR（Virtual Data Room）から取得した資料を管理するディレクトリです。NN資料やIM資料とは別に、VDRから取得した追加資料を保存します。

## ディレクトリ構成
- `nn/`: NN関連VDR資料
- `im/`: IM関連VDR資料
- `memo/`: メモ・補足資料

## 用途
以下のような資料を保存します：
- VDRから取得した追加資料
- 補足資料
- メモ・ノート
- その他、VDR関連の資料

## 使用方法
必要に応じて、VDRから取得した資料をこのディレクトリに配置してください。
