# エージェント関連ディレクトリ

## 概要
エージェント関連のプログラムや案件DD・ロジックを指南するプロンプト関連をまとめたディレクトリです。

## ディレクトリ構成

### prompts/
メタプロンプト集
- `nn_dd/`: NN DD用プロンプト
- `im_dd/`: IM DD用プロンプト
- `additional_dd/`: 追加DD用プロンプト

### workflows/
ワークフロー定義
- `nn_dd_workflow.md`: NN DDワークフロー
- `im_dd_workflow.md`: IM DDワークフロー
- `additional_dd_workflow.md`: 追加DDワークフロー

### commands/
擬似コマンド定義
- コマンド一覧と使用方法

## 使用方法
各ワークフローに従って、対応するプロンプトを使用します。
