# PEファンドDDロジック参考文献

## 概要
PEファンドのデューデリジェンス（DD）ロジックに関する文献を蓄積するディレクトリです。

## 目的
- PEファンドのDDプロセスにおける標準的な評価手法の理解
- 定量情報と定性情報を統合した結論導出ロジックの参考資料
- 業界ベストプラクティスの収集

## 文献の種類

### 1. 学術論文
- PEファンドのDDプロセスに関する学術研究
- 投資判断における定量・定性分析の統合手法

### 2. 業界レポート
- PEファンド協会等が発行するガイドライン
- 業界標準のDDチェックリスト

### 3. Web記事・ブログ
- PEファンドの実務者による知見共有
- ケーススタディや事例分析

### 4. 書籍・書籍の抜粋
- PEファンドのDDに関する専門書籍
- 投資判断に関する理論書

## 文献の追加方法

1. Webから文献を取得
2. 適切なファイル名で保存（例: `pe_dd_guideline_2024.md`）
3. 文献のメタデータを記録：
   - タイトル
   - 著者・発行元
   - 発行日
   - URL（Web記事の場合）
   - 主要な内容の要約

## 文献の活用

### nn_dd/im_ddフェーズでの活用
- 定量情報と定性情報を統合した結論導出ロジックの参考
- 評価基準の設定の参考
- 計算ロジックの設計の参考

### 追加DDフェーズでの活用
- 追加で調査すべき論点の参考
- 議論テーマの設定の参考

## 文献の整理

文献は以下のように分類して整理することを推奨します：

```
references/
├── academic/          # 学術論文
├── industry/          # 業界レポート
├── web/              # Web記事・ブログ
└── books/            # 書籍・書籍の抜粋
```

## 注意事項

- 著作権に注意し、適切に引用する
- Web記事の場合は、URLとアクセス日を記録する
- 文献の内容は定期的に見直し、最新の知見を反映する
