# NN DD計算ロジック

## 概要
NN DD評価に使用する計算ロジックです。定量情報と定性情報を統合した結論導出ロジックを含みます。

## 計算項目

### 1. 財務指標の計算
- 売上成長率（CAGR）
- EBITDAマージン
- 営業利益率
- ROE/ROA

### 2. バリュエーション指標
- EV/Sales倍率
- EV/EBITDA倍率
- PBR倍率
- PER倍率

### 3. スコアリング
- 各論点のスコア化
- 総合スコアの算出
- リスクスコアの算出

## 実装方法
各計算ロジックは、Pythonスクリプトまたはプロンプトベースで実装可能です。

## 出力形式
計算結果は構造化されたJSON形式で出力されます。
