# NN DD評価基準

## 概要
Non Name Sheet（NN）を基にしたNN DD評価の基準です。

## 評価論点

### 1. 事業概要
- 事業内容の明確性
- 市場規模・成長性
- ビジネスモデル

### 2. 財務情報
- 売上規模
- 成長率
- 収益性（EBITDA、営業利益率等）
- 財務健全性

### 3. 経営陣・組織
- 経営陣の経験・実績
- 組織体制
- 人材構成

### 4. 市場・競合
- 市場ポジション
- 競合優位性
- 市場成長性

### 5. 投資条件
- 投資規模
- バリュエーション
- 投資期間
- 出口戦略

## 評価方法
各論点について、定量的・定性的な評価を行います。
