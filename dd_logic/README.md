# DDプロセスロジック

## 概要
PEファンドのデューデリジェンス（DD）プロセスにおける細かいロジックを蓄積するディレクトリです。

## ディレクトリ構成

### nn_dd/
NN DDロジック（Non Name Sheet用）
- `criteria/`: 評価基準
- `calculations/`: 計算ロジック（定量・定性情報を統合した結論導出ロジック含む）
- `outputs/`: 出力フォーマット

### im_dd/
IM DDロジック（Information Memorandum用）
- `evaluation_points/`: 評価論点
- `calculations/`: 計算ロジック（定量・定性情報を統合した結論導出ロジック含む）
- `outputs/`: 出力フォーマット

### additional_dd/
追加DD用フォーマット
- `templates/`: 議論・調査用テンプレート

### references/
PEファンドDDロジックに関する文献
- Webから取得したPEファンドのDDロジックに関する文献を蓄積
- 定量・定性情報を統合した結論導出ロジックの参考資料

## 使用方法
各ワークフローで、対応するロジックを参照・実行します。
