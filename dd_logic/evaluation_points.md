# 評価論点フレームワーク - 仕様書

## 概要

本ドキュメントは、事業承継ファンド向け評価論点管理システムの仕様書です。定量論点と定性論点を体系的に管理し、事業承継投資における意思決定をサポートします。

## 1. 評価論点の構造

### 1.1 論点カテゴリ

評価論点は以下の2つのカテゴリに分類されます：

- **定量論点（Quantitative Points）**: 財務数値や指標に基づく客観的な評価項目
- **定性論点（Qualitative Points）**: 経営陣、組織、事業環境など質的な評価項目

### 1.2 論点の属性

各論点は以下の属性を持ちます：

- **category**: 論点カテゴリ（'quantitative' または 'qualitative'）
- **weight**: 重み（0.0〜1.0）
- **sub_points**: サブ論点のリスト
- **data_sources**: データソースのリスト
- **critical**: クリティカルフラグ（True: クリティカル、False: 非クリティカル）

## 4. スコア計算式

### 4.1 カテゴリスコアの計算

各カテゴリ（定量論点または定性論点）のスコアは、以下の加重平均で計算されます：

$$Score_{category} = \frac{\sum_{i} w_i \times score_i}{\sum_{i} w_i}$$

ここで：
- $Score_{category}$: カテゴリスコア（0-100点）
- $w_i$: 論点$i$のウェイト
- $score_i$: 論点$i$のスコア（0-100点）

### 4.2 総合スコアの計算

総合スコアは、定量論点と定性論点の加重平均で計算されます：

$$Total\ Score = w_{quant} \times Score_{quant} + w_{qual} \times Score_{qual}$$

ここで：
- $Total\ Score$: 総合スコア（0-100点）
- $w_{quant}$: 定量論点のウェイト（NN DD: 0.55、IM DD: 0.55）
- $Score_{quant}$: 定量論点のスコア（0-100点）
- $w_{qual}$: 定性論点のウェイト（NN DD: 0.45、IM DD: 0.45）
- $Score_{qual}$: 定性論点のスコア（0-100点）

### 4.3 クリティカル論点の扱い

クリティカル論点（$critical = True$）は、通常の論点と同様にウェイトに基づいて計算されますが、実務上は重点的に評価する必要があります。

## 5. 評価フレームワークの使用方法

### 5.1 論点の取得

評価論点は、ワークフロータイプ（'nn_dd' または 'im_dd'）とカテゴリ（'quantitative' または 'qualitative'）を指定して取得します。

### 5.2 クリティカル論点の抽出

クリティカル論点のみを抽出して、重点的に評価します。

### 5.3 スコア計算

各論点にスコア（0-100点）を付与し、カテゴリスコアと総合スコアを計算します。

### 5.4 チェックリスト生成

評価論点からチェックリストを生成し、評価漏れを防ぎます。

## 6. 計算例

### 6.1 NN DD評価例

#### 入力データ

定量論点のスコア：
- 財務実績: 85点
- 財務健全性: 90点
- 市場規模: 75点
- 暫定バリュエーション: 80点

定性論点のスコア：
- 事業承継背景: 90点
- ビジネスモデル: 80点
- 経営陣・組織: 65点
- 事業関係性: 75点
- 投資適合性: 85点

#### 計算プロセス

**定量論点スコア**:

$$Score_{quant} = \frac{0.20 \times 85 + 0.15 \times 90 + 0.10 \times 75 + 0.10 \times 80}{0.20 + 0.15 + 0.10 + 0.10}$$

$$= \frac{17.0 + 13.5 + 7.5 + 8.0}{0.55} = \frac{46.0}{0.55} \approx 83.6\ 点$$

**定性論点スコア**:

$$Score_{qual} = \frac{0.15 \times 90 + 0.10 \times 80 + 0.10 \times 65 + 0.05 \times 75 + 0.05 \times 85}{0.15 + 0.10 + 0.10 + 0.05 + 0.05}$$

$$= \frac{13.5 + 8.0 + 6.5 + 3.75 + 4.25}{0.45} = \frac{36.0}{0.45} = 80.0\ 点$$

**総合スコア**:

$$Total\ Score = 0.55 \times 83.6 + 0.45 \times 80.0 = 45.98 + 36.0 = 81.98\ 点$$
