# NN DD結果

## 概要
Non Name Sheet（NN）を基にしたNN DD評価の結果を管理するディレクトリです。NN DDワークフロー実行時に自動生成され、このディレクトリに保存されます。

## ファイル構成
- `report.md`: AIによるNN DD結果レポート（人間が読みやすい形式）
  - 評価結果のサマリー
  - 各論点の評価結果
  - 総合スコアと投資推奨度
  - 最終判定（見送り/見送りでない）

## 評価内容
NN DDでは以下の評価が実施されます：
- **定量情報と定性情報を統合した評価**
- **各論点の評価**:
  - 事業概要
  - 財務情報
  - 経営陣・組織
  - 市場・競合
  - 投資条件
- **スコア算出**:
  - 総合スコア
  - リスクスコア
  - 投資推奨度の判定

## 使用方法

### 1. 入力資料の配置
NN関連VDR資料を `../../vdr/nn/` に配置します。

### 2. ワークフローの実行
NN DDワークフローを実行します。
- 詳細な手順: `.cursor/skills/pe-dd-nn/workflow.md` を参照
- AI/LLMを使用したプロンプトベースで実装
- 計算ロジックは数式仕様書（`dd_logic/nn_dd/calculations/`）を参照

### 3. 結果の確認
結果がこのディレクトリに自動的に生成されます：
- `report.md`: 評価結果レポートを確認

## 評価基準
詳細な評価基準は以下を参照してください：
- `dd_logic/nn_dd/criteria/evaluation_criteria.md`: NN DD評価基準

- **入力資料**: `../../vdr/nn/` - NN関連VDR資料
- **評価ロジック**: `dd_logic/nn_dd/` - NN DD評価ロジック
