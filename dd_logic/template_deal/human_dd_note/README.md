# 人間によるDDノート

## 概要
人間によるDDノートを管理するディレクトリです。AIによる自動評価結果を踏まえ、人間による追加の議論や調査、最終判断を記録します。

## 用途
以下のような内容を保存します：
- 人間による議論・追加調査の結果
- AI DD結果に対する人間による評価・判断
- 専門家による追加評価
- 最終的な投資判断の記録
- その他、人間によるDDプロセスの記録

## 使用方法

### 1. DDノートの作成
AI DD結果を踏まえ、必要に応じて追加の議論や調査を実施し、ノートを作成します。

### 2. 結果の保存
DDノートをこのディレクトリに保存します。
- 議論テーマの設定
- 調査・議論の実施
- 結果の整理
- 最終判断の記録

### 3. ワークフロー
議論・調査の結果を整理し、このディレクトリに保存します。
- `dd_logic/additional_dd/templates/`: 議論用テンプレート（存在する場合）
