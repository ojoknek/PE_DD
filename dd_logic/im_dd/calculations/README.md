# IM DD計算ロジック

## 概要
IM DD評価に使用する計算ロジックです。定量情報と定性情報を統合した結論導出ロジックを含みます。

## 計算項目

### 1. 財務分析
- DCF（Discounted Cash Flow）分析
- コンプレッション分析
- 財務比率分析
- 感度分析

### 2. バリュエーション分析
- 複数のバリュエーション手法の適用
- バリュエーション範囲の算出
- 妥当性の検証

### 3. リスク分析
- リスク要因の定量化
- リスクスコアの算出
- シナリオ分析

### 4. スコアリング
- 各論点のスコア化
- 総合スコアの算出
- 投資判断スコアの算出

## 実装方法
各計算ロジックは、Pythonスクリプトまたはプロンプトベースで実装可能です。

## 出力形式
計算結果は構造化されたJSON形式で出力されます。
